Phishing-Website-Detection

Type: Machine Learning Project

Technologies: Python, scikit-learn, Pandas, NumPy, Matplotlib, UCI Dataset

Proiect de machine learning pentru detectarea automată a site-urilor de phishing folosind modele de clasificare. Ensemble models Random Forest și Gradient Boosting oferă un echilibru între acuratețe și timp de antrenare.
