MatchMate – Microservices Architecture App

Tip: Cloud Web App

Tehnologii: Node.js, JavaScript, Docker, Azure Container Apps, REST API, Azure SQL, MSSQL

Descriere: O aplicație web bazată pe o arhitectură cu microservicii, concepută pentru a potrivi utilizatorii în funcție de preferințele lor de jocuri. Sistemul include trei servicii independente:

  • user-service – pentru înregistrare și autentificare;
  • matchmaking-service – pentru calcularea compatibilității dintre utilizatori și gestionarea cererilor de prietenie;
  • chat-service – pentru mesagerie și feedback.

Fiecare serviciu este containerizat folosind Docker și este implementat în Microsoft Azure prin intermediul Azure Container Apps. Comunicarea între servicii se realizează prin API-uri RESTful.

Scorurile de compatibilitate sunt calculate pe baza intereselor comune, stilului de joc, disponibilității și ratingurilor utilizatorilor. Feedback-ul și istoricul conversațiilor sunt stocate și analizate pentru a îmbunătăți experiența utilizatorilor și moderarea.

Arhitectura promovează modularitatea, scalabilitatea și ușurința în mentenanță.
