MatchMate – Microservices Architecture App

Type: Cloud Web App

Technologies: Node.js, JavaScript, Docker, Azure, REST API, SQL

Aplicație bazată pe microservicii pentru potrivirea utilizatorilor în funcție de preferințele de gaming. Include servicii independente pentru utilizatori, matchmaking și chat, containerizate și deploy-ate în Azure.
